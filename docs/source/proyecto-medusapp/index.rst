El proyecto MedusApp
====================================

Medusapp es una aplicación que permite conducir y controlar una licitación o compulsa de precios.
Esta pensado para que municipios (o secretarías, ministerios, organizaciones, etc) realicen estos procesos de manera transparente al ciudadano, las empresas competidoras.
