El proceso completo de una licitación¶

Este proceso comienza con el alta de un Gobierno (u otro ente oficial) en el sistema. Esto es responsabilida de los administradores de la aplicación y puede solicitarse desde el sitio web de la plataforma MedusApp. Agregar gobiernos.

Esta alta incluye a uno o más usuarios con permisos de cursas los diferentes momentos que conduce. Agregar usuarios.

Este nuevo usuario del sistema ya puede entonces acceder al sistema con las credenciales (usuario y clave) creadas por los administradores de la aplicación.
